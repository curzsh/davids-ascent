# David's Ascent

A **bullet heaven** (Vampire Survivors-like) where you play as David, surviving waves of biblical enemies across 5 curated stages — from lions in the wilderness to the ultimate showdown with Goliath.

Short scripture and dialogue between stages bring the story to life for a church/youth audience.

## Game

- **Genre**: Bullet Heaven / Survivor-like
- **Engine**: [Valthorne](https://github.com/tehnewb/Valthorne) (Java / LWJGL)
- **Platform**: PC
- **Session Length**: ~20-25 minutes per run

## Stages

| Stage | Trial | Enemies | Scripture |
|-------|-------|---------|-----------|
| 1 | The Lion | Lions, wolves, snakes | Psalm 23:1 |
| 2 | The Bear | Bears, wild beasts | 1 Samuel 17:37 |
| 3 | The Scouts | Philistine scouts, archers | 1 Samuel 17:47 |
| 4 | The Army | Soldiers, shieldbearers | 2 Samuel 3:1 |
| 5 | Goliath | Goliath + guards | 1 Samuel 17:45 |

## How to Play

- **WASD / Arrow Keys** — Move David
- **Weapons fire automatically** at nearby enemies
- Kill enemies to drop **XP gems** — collect them to level up
- On level up, **click a card** to pick an upgrade
- Survive all 5 stages and defeat Goliath!

## Building & Running

Requires **Java 21+** and the Gradle wrapper is included.

```bash
# Run the game
./gradlew run

# Build a JAR
./gradlew jar
```
